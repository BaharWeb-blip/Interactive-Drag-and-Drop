# 🖱️ Drag and Drop Project

A simple and interactive Drag and Drop application built with **HTML, CSS, and JavaScript**. This project demonstrates how to use the **HTML5 Drag and Drop API** to move items between different containers with a smooth user experience.

## 🚀 Features

- 🖱️ Drag items easily
- 📦 Drop items into different containers
- ⚡ Smooth drag-and-drop functionality
- 🎨 Clean and responsive user interface
- 💻 Beginner-friendly JavaScript project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)




## 📂 Project Structure


Drag-and-Drop/
│
├── index.html
├── style.css
├── script.js



## 🎯 How It Works

1. Select an item by clicking and dragging it.
2. Move the item to another container.
3. Drop the item into the desired location.
4. Repeat to organize items as needed.
